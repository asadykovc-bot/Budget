# Budget PWA

A static phone-first PWA for weekly personal budget tracking, reserve savings, and one combined self-credit balance.

## Features

- Weekly budget from Monday to Sunday
- Expense history for the current week
- Reserve bank from unused weekly budget
- One combined debt balance with weekly payment
- New credit and refinance preview before confirmation
- Reserve payments toward debt
- JSON export and import for backups or phone migration
- Offline cache through a service worker

## Deploy on GitHub Pages

Upload these files to a public GitHub repository and enable GitHub Pages for the repository. The app stores personal data only in the browser on the device, not in GitHub.
